  **SmartHome** is an iOS application built with SwiftUI for monitoring and controlling a smart home system. The app connects to a server that acts as a bridge between the mobile interface and ESP32-based hardware. It displays recent RFID access events, shows fire alarm status, and allows the user to control lighting colors remotely. Sensor and device data are sent through the server and updated in the app in near real time, providing a simple dashboard for home automation and security monitoring.

  **SmartHome** — це iOS-застосунок, створений на SwiftUI для моніторингу та керування системою розумного будинку. Застосунок підключається до сервера, який виступає посередником між мобільним інтерфейсом і апаратною частиною на базі ESP32. У застосунку відображаються останні RFID-події доступу, статус пожежної сигналізації, а також реалізовано віддалене керування кольором освітлення. Дані із сенсорів і пристроїв передаються через сервер та оновлюються у застосунку майже в реальному часі, що дозволяє зручно контролювати систему безпеки й автоматизації будинку.

  **SmartHome** — это iOS-приложение, разработанное на SwiftUI для мониторинга и управления системой умного дома. Приложение подключается к серверу, который выступает связующим звеном между мобильным интерфейсом и аппаратной частью на базе ESP32. В приложении отображаются последние RFID-события доступа, статус пожарной сигнализации, а также доступно удалённое управление цветом освещения. Данные с датчиков и устройств передаются через сервер и обновляются в приложении почти в реальном времени, что позволяет удобно контролировать безопасность и автоматизацию дома.
